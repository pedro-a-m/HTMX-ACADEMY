Estamos animados para apresentar a você uma poderosa funcionalidade do HTMX que permite associar eventos JavaScript a elementos HTML de forma declarativa: o atributo `hx-on`. Com `hx-on`, você pode facilmente definir quais eventos devem disparar requisições AJAX ou outras ações, proporcionando um controle fino sobre a interatividade em suas aplicações web. Vamos explorar como essa funcionalidade pode ser utilizada para criar uma experiência de usuário mais dinâmica e responsiva.

O que é o atributo `hx-on`?

O `hx-on` é um atributo do HTMX que permite associar eventos JavaScript a elementos HTML, de maneira declarativa, diretamente no markup. Isso significa que você pode especificar eventos como cliques, mudanças de valor, carregamento de elementos e muito mais, sem a necessidade de escrever código JavaScript separado para adicionar event listeners.
